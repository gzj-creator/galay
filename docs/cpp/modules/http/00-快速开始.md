# 00-快速开始

本页只保留与当前仓库真实文件、真实 target、真实公开头一致的最短路径。

## 前置要求

- CMake `>= 3.22`
- `C++23` 编译器
- `galay-kernel >= 5.2.0`
- 启用 TLS 时额外需要 `galay-ssl >= 2.1.0` 及其底层 TLS provider

## 基础构建

下面命令对应根 `CMakeLists.txt` 的真实选项；它会生成库、示例、测试与 benchmark：

下文统一用 `<galay-install-prefix>` 表示你的 galay 安装前缀或 monorepo 共享构建前缀。

```bash
cmake -S . -B build \
  -DCMAKE_BUILD_TYPE=Release \
  -DCMAKE_PREFIX_PATH="<galay-install-prefix>" \
  -DBUILD_EXAMPLES=ON \
  -DBUILD_TESTING=ON \
  -DBUILD_BENCHMARKS=ON \
  -DBUILD_MODULE_EXAMPLES=OFF
cmake --build build --parallel
```

## TLS 构建

如果你要使用 `HttpsServer`、`HttpsClient`、`WssClient`、`H2Server` 或 `H2Client`，必须在配置阶段启用 TLS：

```bash
cmake -S . -B build-ssl \
  -DCMAKE_BUILD_TYPE=Release \
  -DCMAKE_PREFIX_PATH="<galay-install-prefix>" \
  -DGALAY_BUILD_SSL=ON \
  -DBUILD_EXAMPLES=ON \
  -DBUILD_TESTING=ON \
  -DBUILD_BENCHMARKS=ON \
  -DBUILD_MODULE_EXAMPLES=OFF
cmake --build build-ssl --parallel
```

示例证书可直接使用仓库自带的 `test/test.crt` 与 `test/test.key`。

## 第一个 HTTP 示例

源码与 target：

- 服务端：`examples/cpp/http/include/e1_echo_server.cc` → `example_http_include_echo_server`
- 客户端：`examples/cpp/http/include/e2_echo_client.cc` → `example_http_include_echo_client`

运行：

```bash
./build/examples/cpp/http/example_http_include_echo_server 8080
./build/examples/cpp/http/example_http_include_echo_client http://127.0.0.1:8080/echo "hello"
```

## 第一个 h2c 示例

源码与 target：

- 服务端：`examples/include/e9_h2cecho.cpp` → `E9-H2cEchoServer`
- 客户端：`examples/include/e10_h2cecho.cpp` → `E10-H2cEchoClient`

运行：

```bash
./build/examples/E9-H2cEchoServer 9080
./build/examples/E10-H2cEchoClient 127.0.0.1 9080
```

## 第一个 TLS 示例

源码与 target：

- HTTPS 服务端：`examples/include/e5_https.cpp` → `E5-HttpsServer`
- HTTPS 客户端：`examples/include/e6_https.cpp` → `E6-HttpsClient`

运行：

```bash
./build-galay-ssl/examples/E5-HttpsServer 8443 test/test.crt test/test.key
./build-galay-ssl/examples/E6-HttpsClient https://127.0.0.1:8443/
```

## 常用校验 target

测试与 benchmark target 名都直接继承真实文件名：

```bash
cmake --build build --target T26-h2c_server T25-h2c_client B1-HttpServer --parallel 4
cmake --build build-ssl --target T21-https_server T22-https_client T28-h2_server T29-h2_client --parallel 4
```
